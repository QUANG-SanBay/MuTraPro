## 🚀 Cách chạy dự án
### ====== Mới clone về ======
### 1. Cài đặt frontEnd (React)
#### Lưu ý phải cài Node.js
```bash
cd frontEnd
npm install
npm start
```
#### 2.cài đặt backEnd (spring boot)
```bash
cd backEnd
./mvnw spring-boot:run
```
hoặc
```bash
cd backEnd
mvnw spring-boot:run
```
